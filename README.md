# 🧠 EduMind AI

## AI-Powered Personalized Learning Platform

EduMind AI is an intelligent adaptive learning platform designed to provide personalized educational experiences based on individual learning preferences. The system leverages Large Language Models (LLaMA 3.3 via Groq API) to generate customized learning content, interactive quizzes, flashcards, mind maps, summaries, and AI-powered tutoring assistance.

The platform identifies a student's learning style (Visual, Auditory, or Read-Write) and adapts educational content accordingly to improve understanding, retention, and academic performance.

### ✨ Key Features

* 🧬 Learning Style Detection (Visual, Auditory, Read-Write)
* 📖 AI-Powered Topic Explanation
* 📝 Smart Notes Summarization
* 📄 PDF Learning Assistant
* 💬 Interactive AI Tutor
* ❓ Automated Quiz Generation
* 🃏 Flashcard Creation
* 🗺️ Mind Map Generation
* 📊 Performance Analytics Dashboard
* 🏆 Achievement & Progress Tracking
* 🌍 Multilingual Support (English, Hindi, French, German)

### 🛠️ Technology Stack

* Python
* Streamlit
* Groq API
* LLaMA 3.3 70B
* Plotly
* PyPDF2
* ReportLab
* Python Dotenv

### 🎯 Project Objective

The objective of EduMind AI is to bridge the gap between traditional one-size-fits-all learning methods and personalized education by providing AI-driven adaptive learning experiences that cater to individual learning preferences.
